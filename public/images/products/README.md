# Imágenes de Productos

Coloca aquí las imágenes de los electrodomésticos.

## Cómo nombrar los archivos:

Usa el formato: `product-{ID}.{extensión}`

Ejemplos:
- `product-1.jpg` - Para el producto con ID 1
- `product-2.png` - Para el producto con ID 2
- `product-13.jpg` - Para el producto con ID 13

## Formatos soportados:
- `.jpg` o `.jpeg`
- `.png`
- `.webp`

## Ejemplo:

Si tienes un producto con `id: 1` en `products.js`, nombra la imagen como:
```
product-1.jpg
```

El sistema automáticamente intentará cargar esta imagen. Si no existe, usará la imagen de fallback (Unsplash) definida en el archivo de productos.

## Resolución recomendada:
- Mínimo: 800x600px
- Ideal: 1200x900px o superior
- Aspecto: 4:3 (horizontal)
